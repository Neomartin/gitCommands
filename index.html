COMANDOS GIT

ACCIÓN COMMAND DESCRIPTION REF
INICIAR GIT git init Inicializamos el controlador de versiones distribuida GIT en nuestro proyecto
AÑADIR STAGE git status Realiza una verificación del estado actual de mi repositorio con respecto al último stage (git add) de archivos realizado.
BORRAR RAMA git branch -d localBranchName Borra una rama específica. En el caso de la rama no estar mergeada totalmente al main se puede "forzar" usando -D en vez de -d
